# سريع - MVP سوق بيع وشراء في السعودية

تطبيق Mobile-first عربي RTL لبيع وشراء الأشياء بسرعة وبثقة. النسخة الحالية MVP ببيانات تجريبية، ومناسبة كبداية قابلة للتطوير لاحقًا إلى قاعدة بيانات حقيقية.

## التقنية المستخدمة

- React مع Vite
- Tailwind CSS
- lucide-react للأيقونات
- Mock Data داخل `src/data/mockData.js`

## الصفحات المتوفرة

- تسجيل الدخول برقم الجوال والاسم والمدينة
- الصفحة الرئيسية مع البحث، الفئات، والفلاتر
- إضافة إعلان بسرعة مع رفع صورة محلية
- تفاصيل المنتج ومعلومات البائع والحفظ والمراسلة
- قائمة المحادثات وشاشة شات بسيطة
- حسابي: بيانات المستخدم، إعلاناتي، المحفوظات، التقييمات، وتسجيل الخروج

## التشغيل

ثبت الحزم:

```bash
npm install
```

شغل التطبيق:

```bash
npm run dev
```

ثم افتح الرابط الذي يظهر في الطرفية، غالبًا:

```text
http://localhost:5173
```

## البناء للإنتاج

```bash
npm run build
```

## ملاحظات التطوير

- الواجهة عربية بالكامل واتجاهها RTL من ملف `index.html`.
- البيانات الحالية تجريبية ويمكن استبدالها لاحقًا بواجهات API.
- النماذج تضيف الإعلانات داخل حالة التطبيق مؤقتًا فقط.
- يمكن لاحقًا إضافة مصادقة SMS، قاعدة بيانات، رفع صور حقيقي، وإشعارات.
